# LockPattern

Android like lock screen written in Swift.

Generates a hex string of your pattern.

0123

4567

89AB

CDEF

### Libraries Used

* [PureLayout](https://github.com/PureLayout/PureLayout)
* [SwiftHEXColors](https://github.com/thii/SwiftHEXColors)
